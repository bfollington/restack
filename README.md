This is a loose attempt at porting the `VStack` and `HStack` concepts from SwiftUI.

```tsx

  <VStack
    horizontal="center"
    vertical="distribute-edges"
    spacing={10}
  >
    <p>One</p>
    <p>Two</p>
    <p>Three</p>
  </VStack>

```
